# myWeb-calculator
science web calculator (js, css, html) in the same file

Features:

Basic Functions
Addition, subtraction, multiplication, division
Percentage
Parentheses
Delete (DEL) and Clear (C)

Scientific Functions:
Square root
Power (x^y)
Factorial (n!)
Reciprocal (1/x)
π (Pi) and e constant
Logarithm (log base 10)
Natural logarithm (ln)

Trigonometric Functions:
sin(x)
cos(x)
tan(x)

Inverse trigonometric functions (activated via the INV button):
sin⁻¹(x)
cos⁻¹(x)
tan⁻¹(x)

Angle Handling
Normal trig functions convert degrees → radians
